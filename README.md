# Flask + Forgeon

This example shows how to use Flask 3 on Forgeon with Serverless Functions using the [Python Runtime](https://forgeon.io/docs/concepts/functions/serverless-functions/runtimes/python).

## Demo

https://flask-python-template.forgeon.io/

## How it Works

This example uses the Web Server Gateway Interface (WSGI) with Flask to enable handling requests on Forgeon with Serverless Functions.

## Running Locally

```bash
npm i -g forgeon
forgeon dev
```

Your Flask application is now available at `http://localhost:3000`.
